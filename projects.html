Chess Statistics



Github Repo

Website Coming Soon!

Description

This code fetchs user chess data from chess.com using a python get() request. I parse through the result of games, and do necessary adjustments such as finding the boolean results, organizing the pgn score into parsing configuration, and more.

I send the entire formatted pandas dataframe to Postgres where I use f string SQL queries to find specific statistics. I send the results back to python, and graph the results using matplotlib, and print out the various statistical visualization of the user's data.

The graphs show several points of analysis including win/draw/loss by time of day, rating progression over time (overlayed with time controls), and opening choices, both for white and black, and the respective results. There are currently 19 output graphs, and the ability to create several more.
