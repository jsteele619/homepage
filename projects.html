Chess Statistics


Github Repo

Description

This code fetchs user chess data from chess.com, and prints out statistical visualization of the user's data.

The graphs show several points of analysis including win/draw/loss by time of day, rating progression over time (overlayed with time controls), and opening choices, both for white and black, and the respective results.

There are currently 19 output graphs, and the ability to create several more. I will be building out the website in the next weeks.
